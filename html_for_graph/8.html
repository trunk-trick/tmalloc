Fig. 6. Memory offset arithmetic and boundary-sharing behavior within adjacent malloc_chunk structures. Advancing the pointer p pointing to the base of Chunk 1 by its exact size s (calculated via (char*)(p) + s) shifts the address space directly to the start of the next contiguous chunk. Note that the prev_size field of Chunk 2 serves as a shared memory area, capable of holding the tail end of Chunk 1's user payload when Chunk 1 is in an allocated state.
